# Greeting by Language

Aplicação em JavaScript que exibe uma saudação com base no idioma selecionado pelo usuário, utilizando objetos e manipulação do DOM.

## 🚀 Funcionalidades
- Seleção de idioma pelo usuário
- Exibição dinâmica de saudação em diferentes idiomas

## 🛠 Tecnologias
- HTML
- JavaScript
